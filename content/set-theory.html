Set theory

Basic concepts

A set is a collection of elements such that:

  • There is no ordering of the elements.
  • There are no duplicate elements.

A set containing elements $a$, $b$ and $c$ is denoted $\{a,b,c\}$.

Because sets are unordered and free from duplicates, we have that $$\{a,b,c\} = \{c,b,b,a\} $$

The meaning of the symbols $\in, \subset, \subseteq, \cup, \cap$ and related concepts are used in the standard sense and will not be introduced here.

The cardinality of a set $A$ is the number of elements in $A$ and is denoted $|A|$. A set with finite cardinality is said to be a finite set.

The power set $\mathcal{P}(A)$ of a set $A$ is the set of all subsets of $A$.

A tuple is an ordered list of elements. An $n$-tuple is a tuple of $n$ elements.

A tuple containing elements $a$, $b$ and $c$ (in this order) is denoted $(a,b,c)$.

Because tuples are ordered and may contain duplicates, we have that $$(a,b,c) \ne (a,c,b) \quad\text{and}\quad (a,b,c) \ne (a,b,b,c).$$

The cartesian product of two sets $A$ and $B$ is the set of all tuples $(a, b)$ for all combinations of $a \in A$ and $b \in B$.

In [set builder notation] we write this as $$A × B \coloneqq \{(a,b) : a \in A, b \in B\}.$$

We write $A^k$ to denote a cartesian product of $k$ copies of $A$, $A \times A \times \cdots \times A$, of which the elements are on the form $(a_1, a_2, \ldots, a_k)$.

Let $A = \{a, b\}$ and $B = \{1, 2, 3\}$, then $$\begin{aligned} A \times B = \big\{ & (a, 1), (a, 2), (a, 3), \\ & (b, 1), (b, 2), (b, 3) \big\}, \\[0.5em] B \times A = \big\{ & (1, a), (1, b), \\ & (2, a), (2, b), \\ & (3, a), (3, b) \big\}. \end{aligned}$$

Relations

Let $A$ and $B$ be sets. A relation on $A \times B$ is a tuple $(A\times B, R)$ where $R \subset A \times B$. Two elements $a\in A$ and $b\in B$ are said to be related by a relation $R$ if $(a, b) \in R$, we write $aRb$ and read “$a$ is related to $b$ (via (relation) $R$)”.

An endorelation on a set $A$ is a relation on $A \times A$.

The converse relation of a relation $R$ on $A \times B$ is the relation $R^T = \{(y, x) : (x, y) \in R\}$ on $B \times A$. Aka. transpose and opposite relation.

Let $A_1, A_2, \ldots, A_n$ be sets. An $n$-ary relation on $A_1 \times A_2 \times \cdots \times A_n$ is a subset of $A_1 \times A_2 \times \cdots \times A_n$.

If it is not specified that a relation is binary or $n$-ary the relation is understood to be binary.

Formally we say that a relation is a tuple $(A \times B, R)$ where $A$ and $B$ are sets and $R$ is a subset of $A \times B$. Similarly, we say that $(A, R)$ is an endorelation if $A$ is a set and $R$ is a subset of $A \times A$.

Two relations $(R_1, A_1 \times B_1)$ and $(R_2, A_2 \times B_2)$ are considered equal if they are equal as tuples. That is, if $A_1 = A_2$, $B_1 = B_2$ and $R_1 = R_2$.

Each relation on $A \times B$ is an element of $\mathcal{P}(A \times B)$ and vice versa. For a finite set $A$ we have $|\mathcal{P}(A) = 2^{|A|}$ [proof]. Thus, assuming $A$ and $B$ to be finite, the number of relations on $A \times B$ is $|\mathcal{P}(A \times B)| = 2^{|A \times B|}$. Similarly, the number of endorelations on a set $A$ is $2^{|A^2|} = 2^{|A|^2}$.

Types of relations

Let $R$ be an endorelation $A$. We then define the following.

$\forall a \in A \, (aR a)$ $\forall a, b \in A \, (aRb \lor bRa)$. Note: Implies reflexivity. $\forall a,b \in A \, (aRb \rightarrow b{R} a)$ $\forall a,b \in A \, (aRb \rightarrow \lnot (bRa))$ $\forall a,b \in A \, (aRb \land bRa \to a = b)$ $\forall a, b, c \in A \, (aRb \land bRc \to aRc)$ reflexive $\land$ symmetric $\land$ transitive

Let $R$ be a [relation] on $A \times B$. We then define the following.

$\forall a_1, a_2 \in A, b \in B \, (a_1Rb \land a_2Rb \to a_1=a_2)$ Aka. injective. $\forall a \in A, b_1, b_2 \in B \, (aRb_1 \land aRb_2 \to b_1=b_2)$ Aka. partial function. $\forall a \in A \;\exists b \in B \, (aRb)$ $\forall b \in B \;\exists a \in A \, (aRb)$ Aka. surjective.

Functions

A function $f: X \to Y$ is a relation $\mapsto$ on $X \times Y$ that is right-unique and left-total. We write $f(x) = y$ to denote $x \mapsto y$. The set $X$ is the domain and $Y$ is the codomain.

Note that the notation $f:X\to Y$ is shorthand for saying that $(X\times Y, f)$ is a relation that is right-unique and left-total. We use the notation

  • $\operatorname{dom}(f) \coloneqq X$
  • $\operatorname{codom}(f) \coloneqq Y$
  • If $X' \subseteq X$ then $f(X') = \{f(x) : x \in X'\}$.
  • If $X = X_1 \times X_2$ and $x = (x_1, x_2) \in X$ then $f(x_1, x_2) = f(x)$ (multivariate function).

Since functions are merely a special class of relations, the concept of injective and surjective function is directly inherited from the definitions of injective and surjective for relations.

As a consequence of [equality of relations] we have that two functions $f_1: X_1 \to Y_1$ and $f_2: X_2 \to Y_2$ are considered equal if $X_1 = X_2$, $Y_1 = Y_2$ and $f_1(x) = f_2(x)$ for all $x \in X_1 = X_2$.

Let $f : A \to B$ and let $A' \subseteq A$, then $f$ restricted to $A'$ is the function $ $f|_{A'}$ : A' \to B; a \mapsto f(a)$.

A function is sometimes called a map (often with additional structure). More generally a function can be seen as a morphism, the name being motivated by the fact that a function “changes the shape” (morphs) it's domain. The notion of morphisms is explored further in and generalized in category theory.

A function is a thing that produces an output when given an input. For any given function and input the output is always the same. Note that one makes a distinction between a function and a functional expression: Let $f: \mathbb{R} \to \mathbb{R}$ and $g: \mathbb{Z} \to \mathbb{R}$ be functions with the functional expression $x \mapsto 2x + 1$, the functions $f$ and $g$ are formally different functions. However, $f$ restricted to $\mathbb{Z}$ is equal to g, written $f|_{\mathbb{Z}} = g$.

Let $f(x) = x^2 + 1$ be a function on $ℝ$ (meaning $f : ℝ \to ℝ$). Then $f(3) = 10$.

A bijection is a function that is injective and surjective.

A function $f: X \to Y$ has an inverse if and only if the converse relation is a function, in which case the converse relation is the inverse function $f^{-1}: Y \to X$. The existence condition is equivalent to requiring that $f$ is bijective.

The number of functions from $X$ to $Y$ is $|Y|^{|X|}$ when $X$ and $Y$ are finite sets.

Each $x \in X$ can be mapped to $|Y|$ different elements. Thus we have $|Y|\cdot|Y|\cdots|Y|$ possible functions, where the product contains $|X$ factors, one for each element of $X$.

The set builder notation is a notational convenience for constructing new sets. Given a set $A$ and a function $f$ with domain $A$ we define the expression $$\{f(a) : a \in A \}$$ to be the set of elements $f(a)$ for all elements $a \in A$. The colon in the set builder notation is read such that. We read the above defined set as “the set of all elements on the form $f(a)$ such that $a \in A$”.

The set builder notation is naturally extended to multiple sets, we write $$\{f(a_1, a_2, \ldots, a_n) : a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n \}$$ to mean the set of elements on the form $f(a_1, a_2, \ldots, a_n)$ for all combinations of $a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n.$

The set of all functions from $X$ to $Y$ is denoted $Y^X$. The notation is motivated by the fact that the number of functions from $X$ to $Y$ is $|Y|^{|X|}$ when $X$ and $Y$ are finite sets.

Sets with order

When we constructed relations we took a set (or a pair of sets) and added some structure to the set, in the form of a relation on the set. This is a very common procedure; taking a set and adding some structure to construct a new mathematical object. We read $(A, S)$ as “the set $A$ is equipped with structure $S$”. This notion is explored further in abstract algebra.

Here we define some common sets with added structure.

A proset (short for preordered set) is a tuple $(A, \leq)$ where $A$ is a set and $\leq$ is a preorder; a binary relation satisfying

  • reflexivity
  • transitivity

A poset (short for partially ordered set) is a tuple $(A, \leq)$ where $A$ is a set and $\leq$ is a partial order; a binary relation satisfying

  • reflexivity
  • antisymmetry
  • transitivity

A totally ordered set is a tuple $(A, \lt)$ where $A$ is a set and $\lt$ is a total order; a binary relation satisfying

  • connex
  • antisymmetry
  • transitivity

The above definitions have corresponding strict versions (strict partial order and strict total order), where all all relations are kept except the reflexive ones.
