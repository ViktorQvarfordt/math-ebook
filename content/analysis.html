Analysis

A space is a set equipped with some structure. The elements of the space are called points.

A metric (aka distance function) $d$ on a set $X$ is a symmetric function $d: X \times X \to \mathbb{R}$ such that for all $x, y, z \in X$ the following holds.

  • $d(x, y) = 0 \iff x = y$ (identity of indiscernibles)
  • $d(x, y) + d(y, z) \ge d(x, z)$ (triangle inequality or subadditivity)

A metric space is a set equipped with a metric. We write $(X, d_X)$ for a metric space with underlying set $X$ and metric $d_X$.

The real numbers $\mathbb{R}$ equipped with $d(x, y) = |x - y|$ is a metric space.

Infinite processes and limits

A sequence is one of the following.

  • A finite sequence is a function defined on $\{1, 2, \ldots, n\}$ for some $n$.
  • An infinite sequence is a function defined on $\mathbb{Z}_+ = \{1, 2, \ldots\}$.

If $a$ is a sequence we use the notation $a_k \coloneqq a(k)$. Similarly, we say that $(a_n)_{n=1}^\infty$ is a sequence. Note that we can shift the indices, in this way we say that also $(a_n)_\mathbb{N}$ is a sequence. If we do not care about the domain of the sequence, we simply write $(a_n)_n$.

Let $(X, d)$ be a metric space. A Cauchy sequence in $X$ is a sequence $(x_n)_\mathbb{N}$ such that $$ \forall \epsilon > 0 \; \exists N \in \mathbb{N} : n, m > N \implies d(x_n, x_m) \lt \epsilon.$$

The limit $L$ of a sequence $(a_n)_n$ is such that $$ \forall \epsilon > 0 \; \exists N \in \mathbb{N} : n > N \implies d(x_n, L) \lt \epsilon.$$ If a sequence has a limit, the sequence is said to be convergent, otherwise divergent.

A metric space is complete if all Cauchy sequences converge to an element in the space.

Let $(X, d_X)$ and $(Y, d_Y)$ be metric spaces. Let $f: X \to Y$ be a function. The limit $L$ of $f$ as $x$ approaches $c$ is denoted by $$ \lim_{x \to c} f(x) = L. $$ The limit satisfies $$ \begin{gathered} \forall \epsilon \gt 0 \; \exists \delta \gt 0 : \forall x \in X : \\ d_X(x, c) \lt \delta \implies d_Y(f(x), L) \lt \epsilon. \end{gathered} $$

The definition of a limit of a sequence is recovered from the definition of a limit of a function by choosing $X = \mathbb{N}$ and $c = \infty$.

A function $f: X \to Y$ is continuous at a point $c$ if $$ \lim_{x\to c} f(x) = f(c). $$ A function is said to be continuous if it is continuous at every point in the domain.

The derivative at $c$ of a function $f: \mathbb{R} \to \mathbb{R}$ is $$ f'(c) = \lim_{h\to 0} \frac{f(x+h)-f(x)}{h}. $$ The function is said to be differentiable at $c$.

A function that is said to be differentiable if it is differentiable at every point in the domain, in this case the derivative of $f$ is viewed as a function.

Repeatedly differentiating a function $k$ times gives rise to the $k$:th derivative, denoted $f^{(k)}$, if it exists.

A function is continuously differentiable the function if differentiable and the derivative is a continuous function.

A function is smooth if it is infinitely continuously differentiable, i.e. if $f^{(k)}$ is continuously differentiable for all $k \in \mathbb{N}$.

The set of $k$ times continuously differentiable functions is denoted $C^k(\mathbb{R})$. The set of smooth functions is denoted $C^\infty(\mathbb{R})$.
