Terminology and nomenclature

A list of important symbols etc will not be listed here since this can be found elsewhere. Throughout the text terms will be defined and introduced before use.

A list of important words that are used to convey technical meaning is given below.

  • auto-: self- (Ancient Greek) [ref]. Think of “auto-matic”, meaning “self-moving”.

  • endo-: within- (Ancient Greek) [ref]. Think of “endo-thermic”, meaning “within-heating” (an endothermic process absorbs heat from its surroundings).

    The distinction between auto- and endo- will become clear when discussing automorphisms and automorphisms.

  • homo-: same (Ancient Greek) [ref].

  • iso-: equal (Ancient Greek) [ref].

    The distinction between homo- and iso- will become clear when discussing homomorphisms and isomorphisms.

  • -morphism: relating to form or shape.

  • mono-: single- or unique-

  • epi-: on top of-

Set theory

Basic concepts

Definition: Set.

A set is a collection of elements such that:

  • There is no ordering of the elements.
  • There are no duplicate elements.

A set containing elements $a$, $b$ and $c$ is denoted $\{a,b,c\}$.

Example.

Because sets are unordered and free from duplicates, we have that $$\{a,b,c\} = \{c,b,b,a\} $$

The meaning of the symbols $\in, \subset, \subseteq, \cup, \cap$ and related concepts are used in the standard sense and will not be introduced here.

Definition: Cardinality.

The cardinality of a set $A$ is the number of elements in $A$ and is denoted $|A|$. A set with finite cardinality is said to be a finite set.

Definition: Power set.

The power set $\mathcal{P}(A)$ of a set $A$ is the set of all subsets of $A$.

Definition: Set builder notation.

The set builder notation is a notational convenience for constructing new sets. Given a set $A$ we define the expression $$\{f(a) : a \in A \}$$ to mean the set of elements on the form $f(a)$ for all elements $a \in A$, where $f$ is some way of constructing a new element from a given element. This notion is later formalized as a function. The colon in the set builder notation is read such that. We read the above defined set as “the set of all elements on the form $f(a)$ such that $a \in A$”.

The set builder notation is naturally extended to multiple sets, we write $$\{f(a_1, a_2, \ldots, a_n) : a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n \}$$ to mean the set of elements on the form $f(a_1, a_2, \ldots, a_n)$ for all combinations of $a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n.$

Definition: Tuple.

A tuple is an ordered list of elements. An $n$-tuple is a tuple of $n$ elements.

A tuple containing elements $a$, $b$ and $c$ (in this order) is denoted $(a,b,c)$.

Example.

Because tuples are ordered and may contain duplicates, we have that $$(a,b,c) \ne (a,c,b) \quad\text{and}\quad (a,b,c) \ne (a,b,b,c).$$

Definition: Cartesian product.

The cartesian product of two sets $A$ and $B$ is given by $$A × B \coloneqq \{(a,b) : a \in A, b \in B\}.$$

We write $A^k$ to denote a cartesian product of $k$ copies of $A$, $A \times A \times \cdots \times A$.

Intuition.
Example.

Let $A = \{a, b\}$ and $B = \{1, 2, 3\}$, then $$\begin{aligned} A \times B = \big\{ & (a, 1), (a, 2), (a, 3), \\ & (b, 1), (b, 2), (b, 3) \big\}. \end{aligned}$$

Relations

Definition: (Binary) Relation.

Let $A$ and $B$ be sets. A relation on $A \times B$ is a tuple $(A\times B, R)$ where $R \subset A \times B$. Two elements $a\in A$ and $b\in B$ are said to be related by a relation $R$ if $(a, b) \in R$, we write $aRb$ and read “$a$ is related to $b$ (via (relation) $R$)”.

Definition: Endorelation.

An endorelation on a set $A$ is a relation on $A \times A$.

Definition: Converse relation.

The converse relation of a relation $R$ on $A \times B$ is the relation $R^T = \{(y, x) : (x, y) \in R\}$ on $B \times A$. Aka. transpose and opposite relation.

Definition: $n$-ary Relation.

Let $A_1, A_2, \ldots, A_n$ be sets. An $n$-ary relation on $A_1 \times A_2 \times \cdots \times A_n$ is a subset of $A_1 \times A_2 \times \cdots \times A_n$.

If it is not specified that a relation is binary or $n$-ary the relation is understood to be binary.

Remark: Equality of relations.

Formally we say that a relation is a tuple $(A \times B, R)$ where $A$ and $B$ are sets and $R$ is a subset of $A \times B$. Similarly, we say that $(A, R)$ is an endorelation if $A$ is a set and $R$ is a subset of $A \times A$.

Two relations $(R_1, A_1 \times B_1)$ and $(R_2, A_2 \times B_2)$ are considered equal if they are equal as tuples. That is, if $A_1 = A_2$, $B_1 = B_2$ and $R_1 = R_2$.

Observation: Number of relations.

Each relation on $A \times B$ is an element of $\mathcal{P}(A \times B)$ and vice versa. For a finite set $A$ we have $|\mathcal{P}(A) = 2^{|A|}$ [proof]. Thus, assuming $A$ and $B$ to be finite, the number of relations on $A \times B$ is $|\mathcal{P}(A \times B)| = 2^{|A \times B|}$. Similarly, the number of endorelations on a set $A$ is $2^{|A^2|} = 2^{|A|^2}$.

Types of relations

Definition: Types of endorelations.

Let $R$ be an endorelation $A$. We then define the following.

Definition: Reflexive.
$\forall a \in A \, (aR a)$
Definition: Connex.
$\forall a, b \in A \, (aRb \lor bRa)$. Note: Implies reflexivity.
Definition: Symmetric.
$\forall a,b \in A \, (aRb \rightarrow b{R} a)$
Definition: Asymmetric.
$\forall a,b \in A \, (aRb \rightarrow \lnot (bRa))$
Definition: Antisymmetric.
$\forall a,b \in A \, (aRb \land bRa \to a = b)$
Definition: Transitive.
$\forall a, b, c \in A \, (aRb \land bRc \to aRc)$
Definition: Equivalence relation.
reflexive $\land$ symmetric $\land$ transitive

Let $R$ be a relation on $A \times B$. We then define the following.

Definition: Left-unique.
$\forall a_1, a_2 \in A, b \in B \, (a_1Rb \land a_2Rb \to a_1=a_2)$ Aka. injective.
Definition: Right-unique.
$a \in A, b_1, b_2 \in B \, (aRb_1 \land aRb_2 \to b_1=b_2)$ Aka. partial function.
Definition: Left-total.
$\forall a \in A \;\exists b \in B \, (aRb)$
Definition: Right-total.
$\forall b \in B \;\exists a \in A \, (aRb)$ Aka. surjective.

Functions

Definition: Function.

A function $f: X \to Y$ is a relation $\mapsto$ on $X \times Y$ that is right-unique and left-total. We write $f(x) = y$ to denote $x \mapsto y$. The set $X$ is the domain and $Y$ is the codomain.

Note that the notation $f:X\to Y$ is shorthand for saying that $(X\times Y, f)$ is a relation that is right-unique and left-total. We use the notation

  • $\operatorname{dom}(f) \coloneqq X$
  • $\operatorname{codom}(f) \coloneqq Y$
  • If $X' \subseteq X$ then $f(X') = \{f(x) : x \in X'\}$.
  • If $X = X_1 \times X_2$ and $x = (x_1, x_2) \in X$ then $f(x_1, x_2) = f(x)$$ (multivariate function).

Since functions are merely a special class of relations, the concept of injective and surjective function is directly inherited from the definitions of injective and surjective for relations.

Observation: Equality of functions.

As a consequence of [equality of relations] we have that two functions $f_1: X_1 \to Y_1$ and $f_2: X_2 \to Y_2$ are considered equal if $X_1 = X_2$, $Y_1 = Y_2$ and $f_1(x) = f_2(x)$ for all $x \in X_1 = X_2$.

Remark: Wider notions of functions.

A function is sometimes called a map (often with additional structure). More generally a function can be seen as a morphism, the name being motivated by the fact that a function “changes the shape” (morphs) it's domain. The notion of morphisms is explored further in and generalized in category theory.

Intuition: Functions.

A function is a thing that produces an output when given an input. For any given function and input the output is always the same. Note that one makes a distinction between a function and a functional expression: Let $f: \mathbb{R} \to \mathbb{R}$ and $g: \mathbb{Z} \to \mathbb{R}$ be functions with the functional expression $x \mapsto 2x + 1$, the functions $f$ and $g$ are formally different functions. However, $f$ restricted to $\mathbb{Z}$ is equal to g, written $f|_{\mathbb{Z}} = g$.

Example.

Let $f(x) = x^2 + 1$ be a function on $ℝ$ (meaning $f : ℝ \to ℝ$). Then $f(3) = 10$.

Definition: Bijection.

A bijection is a function that is injective and surjective.

Definition: Inverse function.

A function $f: X \to Y$ has an inverse if and only if the converse relation is a function, in which case the converse relation is the inverse function $f^{-1}: Y \to X$. The existence condition is equivalent to requiring that $f$ is bijective.

Observation: Number of functions.

The number of functions from $X$ to $Y$ is $|Y|^{|X|}$ when $X$ and $Y$ are finite sets.

Proof.

Each $x \in X$ can be mapped to $|Y|$ different elements. Thus we have $|Y|\cdot|Y|\cdots|Y|$ possible functions, where the product contains $|X$ factors, one for each element of $X$.

Definition: Set of functions.

The set of all functions from $X$ to $Y$ is denoted $Y^X$. The notation is motivated by the fact that the number of functions from $X$ to $Y$ is $|Y|^{|X|}$ when $X$ and $Y$ are finite sets.

Definition: Set of functions.

The set of all functions from $X$ to $Y$ is denoted $Y^X$. The notation is motivated by the fact that the number of functions from $X$ to $Y$ is $|Y|^{|X|}$ when $X$ and $Y$ are finite sets.

Sets with order

When we constructed relations we took a set (or a pair of sets) and added some structure to the set, in the form of a relation on the set. This is a very common procedure; taking a set and adding some structure to construct a new mathematical object. We read $(A, S)$ as “the set $A$ is equipped with structure $S$”. This notion is explored further in abstract algebra.

Here we define some common sets with added structure.

Definition: Proset.

A proset (short for preordered set) is a tuple $(A, \leq)$ where $A$ is a set and $\leq$ is a preorder; a binary relation satisfying

  • reflexivity
  • transitivity
Definition: Poset.

A poset (short for partially ordered set) is a tuple $(A, \leq)$ where $A$ is a set and $\leq$ is a partial order; a binary relation satisfying

  • reflexivity
  • antisymmetry
  • transitivity
Definition: Totally ordered set.

A totally ordered set is a tuple $(A, \lt)$ where $A$ is a set and $\lt$ is a total order; a binary relation satisfying

  • connex
  • antisymmetry
  • transitivity
Remark.

The above definitions have corresponding strict versions (strict partial order and strict total order), where all all relations are kept except the reflexive ones.

Abstract algebra

Operations and algebraic structures

Definition: Binary operation.

A binary operation on a set $A$ is a function $*: A \times A \to B$ where $A \subseteq B$. We use the infix notation $a*b$ to denote $*(a,b)$.

Let $(A, *)$ be a binary operation. We then define the following properties of $*$.

Definition: Associative.

$\forall a, b, c \in A \, (a*(b*c) = (a*b)*c)$

Intuition.

The operations addition and multiplication on $\mathbb{R}$ are associative. However, subtraction an division are not associative.

Example: Subtraction is not associative.

Let $* : \mathbb{R} \times \mathbb{R} \to \mathbb{R}, (a, b) \mapsto a - b$. Let $c \neq 0$, then $$ \begin{aligned} a*(b*c) &= a - (b - c) \\ &= a - b + c \\ &\ne (a - b) - c \\ &= (a*b)*c. \end{aligned} $$

Definition: Commutative.

$\forall a, b \in A \, (a*b = b*a)$

Definition: Identity element.

An element $e \in A$ is an identity element if it satisfies both:

  • $\forall a \in A \, (e*a = a)$ (left identity)
  • $\forall a \in A \, (a*e = a)$ (right identity)
Proposition: Uniqueness of identity.

If a binary operation has an identity element, then it is unique.

Proof.

Let $(A, *)$ be a binary operation. Assume that both $e$ and $f$ are identity elements of $*$. Then,

  1. $e * f = e$, because $e$ is a left identity.
  2. $e * f = f$, because $f$ is a right identity.
  3. Since the left hand sides of the above equations are equal, also the right hand sides must be equal, thus $e = f$.
Definition: Closure.

Let $* : A \times A \to B$ be a binary operation.

  • We say that $*$ is closed if $*(A) \subseteq A$.
  • Let $A' \subseteq A$. We say that $A'$ has closure under $*$, or that $*$ is closed on $A'$, if $*(A') \subseteq A'$.
Intuition.

Let $*$ be a binary operation on $\mathbb{Z}$ given by $a*b = a - b$. This operation is not closed on $\mathbb{Z}^+$ because there exists $a, b \in \mathbb{Z}^+$ such that $a - b \not\in \mathbb{Z}^+$ (for example $a = 0, b = 1$). On the other hand the binary operation $(\mathbb{Z}, +)$ can be restricted to a binary operation on $\mathbb{Z}^+$.

Definition: Magma.

A magma is a a tuple $(A, *)$ where $A$ is a set and $*$ is a closed binary operation on $A$.

Definition: Monoid.

A monoid is a magma $(A, *)$ such that $*$ is associative and has identity.

Definition: Inverse element.

Let $(A, *)$ be a magma with identity element $e$. An element $a^{-1} \in A$ is an inverse of $a \in A$ if it satisfies both:

  • $a^{-1} * a = e$ (left inverse)
  • $a * a^{-1} = e$ (right inverse)
Proposition: Uniqueness of inverse.

Let $(A, *)$ be a monoid, let $a \in A$. If $a$ has an inverse $a^{-1}$, then it is unique.

Proof.

Let $(A, *)$ be a monoid. Let $a \in A$. Assume that both $a^{-1}$ and $b^{-1}$ are inverse elements of $a$. Then,

  1. $e = a^{-1} * a$.
  2. $e = b^{-1} * a$.
  3. The above together with uniqueness of identity implies $a^{-1} * a = b^{-1} * a$.
  4. Operating with $a^{-1}$ from right gives $(a^{-1} * a) * a^{-1} = (b^{-1} * a) * a^{-1}$.
  5. By associativity $(a^{-1} * a) * a^{-1} = (b^{-1} * a) * a^{-1} \implies a^{-1} * (a * a^{-1}) = b^{-1} * (a * a^{-1})$.
  6. By propery of inverse $a^{-1} * e = b^{-1} * e \implies a^{-1} = b^{-1}$.
Remark.

The theorem does not hold if $*$ is not associative [ref].

Definition: Group.

A group is a monoid $(A, *)$ such that each element of $A$ has an inverse element under $*$.

Definition: Compatibility of operations.

Let $*$ and $\star$ be two binary operations on a set $A$, we say that $*$ and $*$ are compatible if

  • $\forall a, b, c \in A \, \big(a * (b \star c) = (a * b) \star c\big)$
Definition: Monoid action.

Let $(A, *)$ be a monoid with identity $e$ and let $X$ be a set. A (left) monoid action of $(A, *)$ on $X$ is a function $\star : A \times X \to X$ such that

  • $\forall x \in X \, (e \star x = x)$ (compatibility of identity).
  • $*$ and $\star$ are compatible.

Let $G$ be a group, then a group action of $G$ on $X$ is a monoid action of $G$ on $X$. (This makes sense because a group is a monoid).

Definition: Distributivity.

Let $*$ and $\star$ be two binary operations on a set $A$, we say that $*$ is distributive over $\star$ if both

  • $\forall a, b, c \in A \, \big(a * (b \star c) = (a * b) \star (a * c)\big)$ (left distributive)
  • $\forall a, b, c \in A \, \big((b \star c) * a = (b * a) \star (c * a)\big)$ (right distributive)
Definition: Abelian.

An algebraic structure $(A, *)$ is abelian or commutative if $*$ is commutative.

Definition: Ring.

A ring $R$ is a 3-tuple $(A, +, \cdot)$ where $A$ is a set and $+, \cdot$ are binary operations on $A$ such that:

  • $(A, +)$ is an abelian group with identity $0_R$.
  • $(A, \cdot)$ is a monoid with identity $1_R$.
  • Multiplication is distributive over addition.

We say that $A$ is the underlying set of $R$ and write $a \in G$ to denote $a \in A$.

Definition: Module.

A module $M$ is a $4$-tuple $(A, R, +, \cdot)$ where $A$ is a set, $R$ is a ring, $+$ (module addition) is a function $A\times A \to A$ and $\cdot$ (scalar multiplication) is function $R \times A \to A$ such that:

  1. $\forall x \in X \, (1_R \cdot x = x)$ (compatibility of identity).
  2. Scalar multiplication is compatible with ring multiplication.
  3. $(A, +)$ is an abelian group.
  4. $\forall r \in R, x, y \in X \, \big(r \cdot (x + y) = r \cdot x + r \cdot y \big)$.
  5. $\forall r, s \in R, x \in X \, \big((r +_R s) \cdot x = r \cdot x + s \cdot x \big)$.

We say that $(A, R, +, \cdot)$ is a module over the ring $R$. A module should be thought of as a ring action on an abelian group. Indeed, the first two requirements are inherited from monoid action for the monoid $(R, \cdot_R)$, the third requirement confirms that the ring is acting on an abelian group and the last two requirements describe the compatibility of the operations of the ring $(R, +_R, \cdot_R)$ and the abelian group $(A, +)$.

Definition: Field.

A field $F$ is a ring $R = (A, +, \cdot)$ such that $(A\setminus\{0\}, \cdot)$ is an abelian group with $1_R$ as unit..

Definition: Vector space.

A vector space is a module over a field.

The novice reader will be confused by the above, therefore we give a self-contained description of the prototypical vector space, the real vector space $ℝ^n$.

Example: The real vector space $ℝ^n$.

Consider the $n$-fold cartesian product $ℝ^n$ of the real numbers $ℝ$. The points in $ℝ^n$ are called vectors. Vector addition is given by component-wise addition; if $x, y \in ℝ^n$ then $$ \begin{aligned} x + y &= (x_1, x_2, \ldots, x_n) + (y_1, y_2, \ldots, y_n) \\ &= (x_1 + y_1, x_2 + y_2, \ldots, x_n + y_n), \end{aligned} $$ and scalar multiplication with $k \in ℝ$ is given by $$ \begin{aligned} kx &= k(x_1, x_2, \ldots, x_n) \\ &= (kx_1, kx_2, \ldots, kx_n). \end{aligned} $$

As an exercise, check that the this indeed is a vector space according to the abstract definition.

Graph theory

Reference book: Grimaldi, Discrete and combinatorial mathematics, 5th edition, Addison-Wesley. Chapters 11–13.

Definition: Graph.

A (undirected) graph is an ordered pair $G = (V,E)$ comprising a set $V$ of vertices or nodes together with a set $E \subseteq V \times V$ of edges.

Definition: Directed graph.

A directed graph is an ordered pair $G = (V,E)$ comprising a set $V$ of vertices together with a set of $E$ edges. An edge $e \in E$ is an ordered pair of two verticies.

Intuition: Graphs.

Graphs can be used to model many types of relations.

Analysis

Definition: Space.

A space is a set equipped with some structure. The elements of the space are called points.

Definition: Metric.

A metric (aka distance function) $d$ on a set $X$ is a symmetric function $d: X \times X \to \mathbb{R}$ such that for all $x, y, z \in X$ the following holds.

  • $d(x, y) = 0 \iff x = y$ (identity of indiscernibles)
  • $d(x, y) + d(y, z) \ge d(x, z)$ (triangle inequality or subadditivity)
Definition: Metric space.

A metric space is a set equipped with a metric. We write $(X, d_X)$ for a metric space with underlying set $X$ and metric $d_X$.

Example.

The real numbers $\mathbb{R}$ equipped with $d(x, y) = |x - y|$ is a metric space.

Infinite processes and limits

Definition: Sequence.

A sequence is one of the following.

  • A finite sequence is a function defined on $\{1, 2, \ldots, n\}$ for some $n$.
  • An infinite sequence is a function defined on $\mathbb{Z}_+ = \{1, 2, \ldots\}$.

If $a$ is a sequence we use the notation $a_k \coloneqq a(k)$. Similarly, we say that $(a_n)_{n=1}^\infty$ is a sequence. Note that we can shift the indices, in this way we say that also $(a_n)_\mathbb{N}$ is a sequence. If we do not care about the domain of the sequence, we simply write $(a_n)_n$.

Definition: Cauchy sequence.

Let $(X, d)$ be a metric space. A Cauchy sequence in $X$ is a sequence $(x_n)_\mathbb{N}$ such that $$ \forall \epsilon > 0 \; \exists N \in \mathbb{N} : n, m > N \implies d(x_n, x_m) \lt \epsilon.$$

Definition: Limit of sequence.

The limit $L$ of a sequence $(a_n)_n$ is such that $$ \forall \epsilon > 0 \; \exists N \in \mathbb{N} : n > N \implies d(x_n, L) \lt \epsilon.$$ If a sequence has a limit, the sequence is said to be convergent, otherwise divergent.

Definition: Completeness.

A metric space is complete if all Cauchy sequences converge to an element in the space.

Definition: Limit of function.

Let $(X, d_X)$ and $(Y, d_Y)$ be metric spaces. Let $f: X \to Y$ be a function. The limit $L$ of $f$ as $x$ approaches $c$ is denoted by $$ \lim_{x \to c} f(x) = L. $$ The limit satisfies $$ \begin{gathered} \forall \epsilon \gt 0 \; \exists \delta \gt 0 : \forall x \in X : \\ d_X(x, c) \lt \delta \implies d_Y(f(x), L) \lt \epsilon. \end{gathered} $$

Remark.

The definition of a limit of a sequence is recovered from the definition of a limit of a function by choosing $X = \mathbb{N}$ and $c = \infty$.

Definition: Continuous function.

A function $f: X \to Y$ is continuous at a point $c$ if $$ \lim_{x\to c} f(x) = f(c). $$ A function is said to be continuous if it is continuous at every point in the domain.

Definition: Derivative.

The derivative at $c$ of a function $f: \mathbb{R} \to \mathbb{R}$ is $$ f'(c) = \lim_{h\to 0} \frac{f(x+h)-f(x)}{h}. $$ The function is said to be differentiable at $c$.

A function that is said to be differentiable if it is differentiable at every point in the domain, in this case the derivative of $f$ is viewed as a function.

Repeatedly differentiating a function $k$ times gives rise to the $k$:th derivative, denoted $f^{(k)}$, if it exists.

Definition: Continuously differentiable.

A function is continuously differentiable the function if differentiable and the derivative is a continuous function.

Definition: Smooth function.

A function is smooth if it is infinitely continuously differentiable, i.e. if $f^{(k)}$ is continuously differentiable for all $k \in \mathbb{N}$.

Definition: Set of differentiable functions.

The set of $k$ times continuously differentiable functions is denoted $C^k(\mathbb{R})$. The set of smooth functions is denoted $C^\infty(\mathbb{R})$.

Topology

Definition: Topology.

A topology $\tau$ on a set $X$ is a collection of subsets of $X$ such that:

  • $\{\}, X \in \tau$
  • $\{x_k\}_k \subset \tau \implies \bigcup_k x_k \in \tau$ (closed under infinite unions)
  • $x_1, x_2 \in \tau \implies x_1 \cap x_2 \in \tau$ (closed under finite intersections)

If $\tau$ is a topology on $X$ the pair $(X, \tau)$ is a topological space.

A subset $A$ of $(X, \tau)$ is said to be

  • open if $A \in \tau$,
  • closed if $A^c \in \tau$.
Definition: Standard topology on $\mathbb{R}$.

The standard topology on $\mathbb{R}$ is given by specifying that open sets are given by [ref]

Definition: Continuous function on topological spaces.

A function $f:X\to Y$ where $X$ and $Y$ are topological spaces is continuous if $f^{-1}(A)$ is open in $Y$ for all open sets $A$ of $X$.
