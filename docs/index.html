Mathematics

An experiment in combining a formal and an intuitive approach to mathematics. Source on GitHub.

This text is designed to be read on a computer, mobile device or e-reader, not on paper. References are hyperlinks, in this way no numbering is needed like when typesetting PDFs with LaTeX. Explanations intended to help develop the intuition can be toggled and hidden as to not distract the more advanced reader. In this way the terse dry formalism is combined with lengthy intuitive explanations in a way that is not possible in a physical book.

Terminology and nomenclature

A list of important symbols etc will not be listed here since this can be found elsewhere. Throughout the text terms will be defined and introduced before use.

A list of important words that are used to convey technical meaning is given below.

  • auto-: self- (Ancient Greek) [ref]. Think of “auto-matic”, meaning “self-moving”.

  • endo-: within- (Ancient Greek) [ref]. Think of “endo-thermic”, meaning “within-heating” (an endothermic process absorbs heat from its surroundings).

    The distinction between auto- and endo- will become clear when discussing automorphisms and automorphisms.

  • homo-: same (Ancient Greek) [ref].

  • iso-: equal (Ancient Greek) [ref].

    The distinction between homo- and iso- will become clear when discussing homomorphisms and isomorphisms.

  • -morphism: relating to form or shape.

  • mono-: single- or unique-

  • epi-: on top of-

Set theory

Basic concepts

Definition: Set.

A set is a collection of elements such that:

  • There is no ordering of the elements.
  • There are no duplicate elements.

A set containing elements aaa, bbb and ccc is denoted {a,b,c}\{a,b,c\}{a,b,c}.

Example.

Because sets are unordered and free from duplicates, we have that {a,b,c}={c,b,b,a}\{a,b,c\} = \{c,b,b,a\} {a,b,c}={c,b,b,a}

The meaning of the symbols ∈,⊂,⊆,∪,∩\in, \subset, \subseteq, \cup, \cap∈,⊂,⊆,∪,∩ and related concepts are used in the standard sense and will not be introduced here.

Definition: Cardinality.

The cardinality of a set AAA is the number of elements in AAA and is denoted ∣A∣|A|∣A∣. A set with finite cardinality is said to be a finite set.

Definition: Power set.

The power set P(A)\mathcal{P}(A)P(A) of a set AAA is the set of all subsets of AAA.

Definition: Set builder notation.

The set builder notation is a notational convenience for constructing new sets. Given a set AAA we define the expression {f(a):a∈A}\{f(a) : a \in A \}{f(a):a∈A} to mean the set of elements on the form f(a)f(a)f(a) for all elements a∈Aa \in Aa∈A, where fff is some way of constructing a new element from a given element. This notion is later formalized as a function. The colon in the set builder notation is read such that. We read the above defined set as “the set of all elements on the form f(a)f(a)f(a) such that a∈Aa \in Aa∈A”.

The set builder notation is naturally extended to multiple sets, we write {f(a1,a2,…,an):a1∈A1,a2∈A2,…an∈An}\{f(a_1, a_2, \ldots, a_n) : a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n \}{f(a1​,a2​,…,an​):a1​∈A1​,a2​∈A2​,…an​∈An​} to mean the set of elements on the form f(a1,a2,…,an)f(a_1, a_2, \ldots, a_n)f(a1​,a2​,…,an​) for all combinations of a1∈A1,a2∈A2,…an∈An.a_1 \in A_1, a_2 \in A_2, \ldots a_n \in A_n.a1​∈A1​,a2​∈A2​,…an​∈An​.

Definition: Tuple.

A tuple is an ordered list of elements. An nnn-tuple is a tuple of nnn elements.

A tuple containing elements aaa, bbb and ccc (in this order) is denoted (a,b,c)(a,b,c)(a,b,c).

Relations

Definition: (Binary) Relation.

Let AAA and BBB be sets. A relation on A×BA \times BA×B is a tuple (A×B,R)(A\times B, R)(A×B,R) where R⊂A×BR \subset A \times BR⊂A×B. Two elements a∈Aa\in Aa∈A and b∈Bb\in Bb∈B are said to be related by a relation RRR if (a,b)∈R(a, b) \in R(a,b)∈R, we write aRbaRbaRb and read “aaa is related to bbb (via (relation) RRR)”.

Definition: Endorelation.

An endorelation on a set AAA is a relation on A×AA \times AA×A.

Definition: Converse relation.

The converse relation of a relation RRR on A×BA \times BA×B is the relation RT={(y,x):(x,y)∈R}R^T = \{(y, x) : (x, y) \in R\}RT={(y,x):(x,y)∈R} on B×AB \times AB×A. Aka. transpose and opposite relation.

Remark: Equality of relations.

Formally we say that a relation is a tuple (A×B,R)(A \times B, R)(A×B,R) where AAA and BBB are sets and RRR is a subset of A×BA \times BA×B. Similarly, we say that (A,R)(A, R)(A,R) is an endorelation if AAA is a set and RRR is a subset of A×AA \times AA×A.

Functions

Definition: Function.

A function f:X→Yf: X \to Yf:X→Y is a relation ↦\mapsto↦ on X×YX \times YX×Y that is right-unique and left-total. We write f(x)=yf(x) = yf(x)=y to denote x↦yx \mapsto yx↦y. The set XXX is the domain and YYY is the codomain.

Note that the notation f:X→Yf:X\to Yf:X→Y is shorthand for saying that (X×Y,f)(X\times Y, f)(X×Y,f) is a relation that is right-unique and left-total. We use the notation

  • dom⁡(f):=X\operatorname{dom}(f) \coloneqq Xdom(f):=X
  • codom⁡(f):=Y\operatorname{codom}(f) \coloneqq Ycodom(f):=Y
  • If X′⊆XX' \subseteq XX′⊆X then f(X′)={f(x):x∈X′}f(X') = \{f(x) : x \in X'\}f(X′)={f(x):x∈X′}.
  • If X=X1×X2X = X_1 \times X_2X=X1​×X2​ and x=(x1,x2)∈Xx = (x_1, x_2) \in Xx=(x1​,x2​)∈X then f(x1,x2)=f(x)f(x_1, x_2) = f(x)f(x1​,x2​)=f(x) (multivariate function).

Since functions are merely a special class of relations, the concept of injective and surjective function is directly inherited from the definitions of injective and surjective for relations.

As a consequence of [equality of relations] we have that two functions f1:X1→Y1f_1: X_1 \to Y_1f1​:X1​→Y1​ and f2:X2→Y2f_2: X_2 \to Y_2f2​:X2​→Y2​ are considered equal if X1=X2X_1 = X_2X1​=X2​, Y1=Y2Y_1 = Y_2Y1​=Y2​ and f1(x)=f2(x)f_1(x) = f_2(x)f1​(x)=f2​(x) for all x∈X1=X2x \in X_1 = X_2x∈X1​=X2​.

Remark: Wider notions of functions.

A function is sometimes called a map (often with additional structure). More generally a function can be seen as a morphism, the name being motivated by the fact that a function “changes the shape” (morphs) it's domain. The notion of morphisms is explored further in and generalized in category theory.

Intuition: Functions.

A function is a thing that produces an output when given an input. For any given function and input the output is always the same. Note that one makes a distinction between a function and a functional expression: Let f:R→Rf: \mathbb{R} \to \mathbb{R}f:R→R and g:Z→Rg: \mathbb{Z} \to \mathbb{R}g:Z→R be functions with the functional expression x↦2x+1x \mapsto 2x + 1x↦2x+1, the functions fff and ggg are formally different functions. However, fff restricted to Z\mathbb{Z}Z is equal to g, written f∣Z=gf|_{\mathbb{Z}} = gf∣Z​=g.

Example.

Let f(x)=x2+1f(x) = x^2 + 1f(x)=x2+1 be a function on RℝR (meaning f:R→Rf : ℝ \to ℝf:R→R). Then f(3)=10f(3) = 10f(3)=10.

Definition: Bijection.

A bijection is a function that is injective and surjective.

Definition: Inverse function.

A function f:X→Yf: X \to Yf:X→Y has an inverse if and only if the converse relation is a function, in which case the converse relation is the inverse function f−1:Y→Xf^{-1}: Y \to Xf−1:Y→X. The existence condition is equivalent to requiring that fff is bijective.

Sets with order

When we constructed relations we took a set (or a pair of sets) and added some structure to the set, in the form of a relation on the set. This is a very common procedure; taking a set and adding some structure to construct a new mathematical object. We read (A,S)(A, S)(A,S) as “the set AAA is equipped with structure SSS”. This notion is explored further in abstract algebra.

Here we define some common sets with added structure.

Definition: Proset.

A proset (short for preordered set) is a tuple (A,≤)(A, \leq)(A,≤) where AAA is a set and ≤\leq≤ is a preorder; a binary relation satisfying

  • reflexivity
  • transitivity
Definition: Poset.

A poset (short for partially ordered set) is a tuple (A,≤)(A, \leq)(A,≤) where AAA is a set and ≤\leq≤ is a partial order; a binary relation satisfying

  • reflexivity
  • antisymmetry
  • transitivity
Definition: Totally ordered set.

A totally ordered set is a tuple (A,<)(A, \lt)(A,<) where AAA is a set and <\lt< is a total order; a binary relation satisfying

  • connex
  • antisymmetry
  • transitivity
Remark.

The above definitions have corresponding strict versions (strict partial order and strict total order), where all all relations are kept except the reflexive ones.

Operations and algebraic structures

Definition: Binary operation.

A binary operation on a set AAA is a function ∗:A×A→B*: A \times A \to B∗:A×A→B where A⊆BA \subseteq BA⊆B. We use the infix notation a∗ba*ba∗b to denote ∗(a,b)*(a,b)∗(a,b).

Let (A,∗)(A, *)(A,∗) be a binary operation. We then define the following properties of ∗*∗.

Definition: Associative.

∀a,b,c∈A (a∗(b∗c)=(a∗b)∗c)\forall a, b, c \in A \, (a*(b*c) = (a*b)*c)∀a,b,c∈A(a∗(b∗c)=(a∗b)∗c)

Intuition.

The operations addition and multiplication on R\mathbb{R}R are associative. However, subtraction an division are not associative.

Example: Subtraction is not associative.

Let ∗:R×R→R,(a,b)↦a−b* : \mathbb{R} \times \mathbb{R} \to \mathbb{R}, (a, b) \mapsto a - b∗:R×R→R,(a,b)↦a−b. Let c≠0c \neq 0c̸​=0, then a∗(b∗c)=a−(b−c)=a−b+c≠(a−b)−c=(a∗b)∗c. \begin{aligned} a*(b*c) &= a - (b - c) \\ &= a - b + c \\ &\ne (a - b) - c \\ &= (a*b)*c. \end{aligned} a∗(b∗c)​=a−(b−c)=a−b+c̸​=(a−b)−c=(a∗b)∗c.​

Definition: Commutative.

∀a,b∈A (a∗b=b∗a)\forall a, b \in A \, (a*b = b*a)∀a,b∈A(a∗b=b∗a)

Definition: Identity element.

An element e∈Ae \in Ae∈A is an identity element if it satisfies both:

  • ∀a∈A (e∗a=a)\forall a \in A \, (e*a = a)∀a∈A(e∗a=a) (left identity)
  • ∀a∈A (a∗e=a)\forall a \in A \, (a*e = a)∀a∈A(a∗e=a) (right identity)
Proposition: Uniqueness of identity.

If a binary operation has an identity element, then it is unique.

Proof.

Let (A,∗)(A, *)(A,∗) be a binary operation. Assume that both eee and fff are identity elements of ∗*∗. Then,

  1. e∗f=ee * f = ee∗f=e, because eee is a left identity.
  2. e∗f=fe * f = fe∗f=f, because fff is a right identity.
  3. Since the left hand sides of the above equations are equal, also the right hand sides must be equal, thus e=fe = fe=f.
Definition: Closure.

Let ∗:A×A→B* : A \times A \to B∗:A×A→B be a binary operation.

  • We say that ∗*∗ is closed if ∗(A)⊆A*(A) \subseteq A∗(A)⊆A.
  • Let A′⊆AA' \subseteq AA′⊆A. We say that A′A'A′ has closure under ∗*∗, or that ∗*∗ is closed on A′A'A′, if ∗(A′)⊆A′*(A') \subseteq A'∗(A′)⊆A′.

This operation is not closed on Z+\mathbb{Z}^+Z+ because there exists a,b∈Z+a, b \in \mathbb{Z}^+a,b∈Z+ such that a−b̸∈Z+a - b \not\in \mathbb{Z}^+a−b̸​∈Z+ (for example a=0,b=1a = 0, b = 1a=0,b=1). A magma is a a tuple (A,∗)(A, *)(A,∗) where AAA is a set and ∗*∗ is a closed binary operation on AAA.

Definition: Monoid.

A monoid is a magma (A,∗)(A, *)(A,∗) such that ∗*∗ is associative and has identity.
